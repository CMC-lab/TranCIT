###########################
Dynamic Causal Strength (DCS)
###########################

**Dynamic Causal Strength (DCS)** is a Python package for detecting and analyzing causal relationships in time series data [cite: dynamic-causal-strength/README.md, dynamic-causal-strength/docs/index.rst].
It provides tools for model estimation, causality detection, and simulation, with a focus on dynamic, non-stationary systems [cite: dynamic-causal-strength/README.md, dynamic-causal-strength/docs/index.rst].
